# Lab Activity 1

Write a program that does the following:

- Include a three line comment at the top of the program that says `DEV 108 Lab Activity 1`, today's date, and your name.
- Output `Hello, World!` to the screen.
- Add two numbers together and output the math equation and the total to the screen (i.e. `10 + 55 = 65`). Have the program do the math calculations.
- Multiply three numbers together and output the math equation and the product to the screen (i.e. `5 x 10 x 2 = 100`). Have the program do the math calculations.
- Create a variable named `firstName` and store your first name to it as a string.
- Create a variable named `lastName` and store your last name to it as a string.
- Create a variable named `major` and store your major to it as a string.
- Output the message "My name is " and print out the string value of the `firstName` and `lastName` variables. Tip: be sure there is a space between "is" and your first and last name. Include a period at the end of the sentence.
- Output the message "My major is " and print out the string value of the `major` variable. Be sure to format the sentence with proper spacing and end it with a period.
- Create an input statement and ask the user to enter a number from 0 to 100 — save the input to a variable named `num1`. Print the variable value to the screen in a sentence such as "You entered 55."
- Create an input statement and ask the user to enter a number from 10 to 10000 — save the input to a variable named `num2`. Print the variable value to the screen in a sentence such as "You entered 99."
- Output a math equation sentence using the variables `num1` and `num2` and display the value of `num1` multiplied by `num2` to the screen, with the answer calculated by the program.
- Experiment with a line or two of code of your own for further Python practice (perhaps more math, or another input statement?).

## Example output
